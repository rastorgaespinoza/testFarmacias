# TestFarmacias

Este es un proyecto que lista las farmacias. Utiliza Alamofire

## Iniciar el proyecto

Luego de clonar, debes asegurarte de instalar los pods. Para instalar, abre un ventana de terminal en la carpeta del proyecto y luego escribe:

```
pod install
```
